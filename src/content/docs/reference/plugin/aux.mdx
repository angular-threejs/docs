---
title: aux
description: angular-three-plugin:aux API
---

`angular-three-plugin:aux` is a generator/schematic that allows you to add `angular-three` auxiliary packages like `angular-three-soba` with their respective peer dependencies.

```bash
npx ng g angular-three-plugin:aux
[NGT] Initialize auxiliary packages
? Which packages to add? …
✔ angular-three-soba
✔ angular-three-rapier
✔ angular-three-postprocessing
✔ angular-three-cannon
✔ angular-three-tweakpane
```
