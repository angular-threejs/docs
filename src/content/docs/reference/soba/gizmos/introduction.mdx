`angular-three-soba/gizmos` provides interactive gizmo components for manipulating camera orientation and transforming 3D objects in your scene.

## Available Components

- [NgtsGizmoHelper](./gizmo-helper) - Orientation helper with viewcube/viewport
- [NgtsTransformControls](./transform-controls) - Object transformation controls
- [NgtsPivotControls](./pivot-controls) - Pivot-based transformation
