A collection of utilities that are built on top of `angular-three` to provide additional functionalities out of the box.

Check out the [Soba Storybook](https://soba.angularthree.org) for interactive examples of some components.

## Installation

<Tabs>
	<TabItem label="ng">```shell ng generate angular-three-plugin:aux ```</TabItem>
	<TabItem label="nx">```shell nx generate angular-three-plugin:aux ```</TabItem>
</Tabs>

then select `angular-three-soba` from the list.
