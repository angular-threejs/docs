`NgtRoutedScene` is a simple component that wraps around [`RouterOutlet`](https://angular.dev/api/router/RouterOutlet) and provides a default layout for routed scenes.

## Usage

```angular-ts
import { NgtCanvas, NgtCanvasContent } from 'angular-three/dom';
import { NgtRoutedScene } from 'angular-three';

@Component({
    template: `
        <ngt-canvas>
            <ngt-routed-scene *canvasContent />
        </ngt-canvas>
    `,
    imports: [NgtRoutedScene, NgtCanvas, NgtCanvasContent]
})
export class MyCmp {}
```
