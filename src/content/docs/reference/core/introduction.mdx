`angular-three` is the core library from the Angular Three ecosystem. It contains the custom [Angular Renderer](https://angular.dev/api/core/Renderer2) that knows how to render [THREE.js](https://threejs.org) objects given a [Canvas](https://developer.mozilla.org/en-US/docs/Web/API/HTMLCanvasElement) implementation.

## Installation

```bash showLineNumbers=false
npm i angular-three three
npm i -D @types/three
```
